# ü©∏ Blood Donation Management System (Full-Stack)

A **full-stack Blood Donation Management System** built using **Node.js, Express, MongoDB, and Vanilla JavaScript**.  
The application enables users to sign up, log in, donate blood, and search for donors based on blood group and location.

This project was developed end-to-end, including backend API development, database design, frontend integration, debugging, and production deployment.
